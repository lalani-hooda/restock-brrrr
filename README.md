# restock-brrrr

## Get Started

### Prerequisites

To run restock-brrrr you will need:

- [Python](https://python.org) 3.7+

### Installation

_Clone the repository or checkout with subversion to get a copy of restock-brrrr_

```bash
$ pip3 install -r requirements.txt
```

### Usage

```bash
$ python3 src/main.py
```
